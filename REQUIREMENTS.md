# CubeLab 需求文件

> 給兒子玩的魔術方塊網頁版。第一版只做「能轉、能打亂、能重置、能偵測完成」的最小可玩版本。

## 目標使用者
小孩。介面要乾淨明亮、童趣但不幼稚，按鈕大且好點。

## 第一版功能（MVP）

### 核心
- 標準 3×3 魔術方塊，6 面標準配色
  - 上 = 白、下 = 黃、右 = 紅、左 = 橙、前 = 綠、後 = 藍
- 滑鼠／觸控拖曳「空白處」可旋轉整個方塊視角
- 點選並拖曳「方塊面」可轉動該層，轉動帶平滑動畫
- 「打亂」按鈕：隨機轉動數十步打亂方塊
- 「重置」按鈕：回到還原狀態
- 偵測是否完成（六面同色），完成時給明顯的視覺回饋（橫幅）

### 介面
- 桌機與手機都能順暢操作（響應式）
- 按鈕大、好點，色彩活潑

### 技術
- 單一 HTML 檔，CDN 載入依賴（Three.js）
- 不需 build step，雙擊就能在瀏覽器跑

## 第一版不做（之後再評估）
- 計時器
- 步數計算
- 成績記錄
- 教學模式
- 還原提示

## 開發節奏
> 先做出能轉、能打亂、能重置的最小可玩版本，確認 OK 再加東西。
> 等兒子實際玩過、給出回饋，再決定第二版加什麼。

## 互動細節
- 拖曳閾值 ~10px 才觸發層轉動，避免誤觸
- 層轉動進行中，禁止其它輸入
- 點到方塊面 → 鎖住視角拖曳；點空白處 → 視角拖曳
- 拖曳方向決定旋轉軸：把手指/滑鼠拉的方向投影到方塊面的兩個切線軸，取較接近者
